在 Core 中创建新 Co-loader

本指南提供在 core.flexport.com 中完整新建一个 Co-loader（共装商）的逐步操作说明，附 UI 截图对照。

系统core.flexport.com
Service Provider 类型Freight Partner → Co-loader
更新日期2026 年 4 月
权限要求core:partner_write
⚠️
前置权限要求您的账号需具备 core:partner_write 权限才能创建或编辑 Service Provider。如没有此权限，请联系您的直属经理或 IT 团队申请。
📋 操作步骤总览
1进入 Network → Service Providers 2点击「Add Service Provider」 3填写 Co-loader 基本信息 4Submit 提交并确认档案 5上传费率表（Rate Ingestion）
1
导航
进入 Network → Service Providers 页面

登录 Core 后，在顶部导航栏点击 Network（深色背景高亮），进入 Service Providers 列表。所有 Co-loader 均以 Freight Partner 类型列示于此。

在 Service Providers 列表页的左上角找到蓝紫色「Add Service Provider」按钮，点击后系统将弹出新建表单。

3

↑ 「Add a New Service Provider」弹窗（基于真实 Core 界面还原）— 红框 A 为关键字段，其余字段按侧边注释填写

各字段填写说明：

字段说明Co-loader 填写值
SERVICE PROVIDER TYPE必填 决定该 Service Provider 在系统内的角色 ✅ 选 Freight Partner（默认已选，勿更改）
COMPANY NAME必填 Co-loader 完整法定公司名，须与合同一致 如：Shipco Transport / ECU Worldwide / AMASS Logistics
COMPANY WEBSITE选填 Co-loader 官方网站 如：https://www.shipco.com
COUNTRIES必填 Co-loader 业务覆盖地区 全球运营选 Global（默认）；区域性合作商选具体国家
SCAC CODE建议填写 Standard Carrier Alpha Code，影响费率 CSV 自动映射 向 Co-loader 索取或查 NMFTA 官网
Is Demo Agent选填 标记为测试账号 ❌ 勿勾选（正式 Co-loader）
⚠️
提交前必查：避免重复创建在 Service Providers 列表搜索该公司名，确认无重复记录。重复创建会导致费率映射混乱，影响发票核对和数据分析。
4
提交
点击「Submit」提交并确认档案创建

所有字段确认无误后，点击弹窗右下角的蓝色 Submit 按钮，系统将自动创建该 Co-loader 的 Service Provider 档案并分配唯一 ID。

↑ 确认填写信息后，点击红框标注的「Submit」按钮完成创建；「Cancel」按钮放弃并关闭弹窗
  • 1
    点击 Submit 后弹窗自动关闭，页面回到 Service Providers 列表。
  • 2
    在列表搜索框输入公司名称，确认新记录已出现，Company Type 列显示「Freight Partner」，即创建成功。
  • 3
    点击进入档案详情页，URL 为 core.flexport.com/partners/<ID>，可在此继续完善 SCAC、Countries 等信息。

5
后续操作
上传 Co-loader 费率表（OCS Rate Manager）

Co-loader 档案创建后，须通过 OCS Rate Manager 上传费率表。只有完成费率上传，系统才能在 Marketplace 生成该 Co-loader 的 LCL 服务 offering，进而用于正式订舱。

↑ OCS Rate Manager（实际截图还原）— 左侧历史任务列表，左上角「+」按钮（红框）用于创建新的上传/下载任务

费率上传步骤（点击「+」后的操作）：

  • ①
    上传前准备（首次上传该 Co-loader 时必做）：
    • CFS to Port Mappings — 确认 Co-loader 港口代码已有对应 Flexport 内部代码
    • Charge Code Mappings — 确认费用项名称已映射至 Flexport 标准费用项
  • ②
    点击 Rate Manager 左上角 + 按钮，弹出 Create a New Task 表单。
  • ③
    Task Type 选 Upload，Content Type 选 Rates，选择对应 Co-loader，填写费率有效期（Valid From / Valid To），上传 CSV 费率文件，点击 Create Task。
  • ④
    系统后台处理任务，完成后在 Slack #tetrominos-job-history 频道收到通知。
  • ⑤
    在 Rate Manager 下载输出 CSV，确认各行状态为 Updated。如有 Failed 行，修正后重新上传完整文件（系统为全量替换逻辑）。

ℹ️
全量替换逻辑：同一 Co-loader 同一有效期内，新上传文件将完全替换已有费率——原有航线若不在新文件中将自动归档（Archived）。因此每次上传须包含该时段内所有有效航线的完整数据。

📋 完整操作流程回顾

1️⃣
进入 Service Providers导航至 core.flexport.com/partners，确认 Network 菜单激活
2️⃣
点击 Add Service Provider左上角蓝紫色按钮，需具备 core:partner_write 权限
3️⃣
填写 Co-loader 信息Type 选 Freight Partner，填写公司名称、SCAC Code、国家范围
4️⃣
Submit 提交创建列表出现 Company Type = Freight Partner 记录即为成功
5️⃣
上传费率表Rate Manager 中新建 Upload Rates 任务，上传 Co-loader CSV 费率文件
✅
全部完成费率上传成功后，Marketplace 生成 LCL offering，可正式用于订舱
